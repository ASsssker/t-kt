## Установка

Запускаем команды
```bash
  git clone https://github.com/ASsssker/t-kt.git
  cd t-kt
  sudo ./install.sh
```
После установки утилиту можно запускать в терминале командой
```bash
  sudo t-kt
```

## Обновление

```bash
  cd t-kt
  git pull
  sudo ./install.sh
```
